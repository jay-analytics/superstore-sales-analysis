# Superstore Sales Analysis

## 📊 Project Overview

This project analyzes sales data to identify key business insights.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📌 Key Insights

* East region has highest profit margin
* Technology category generates highest profit
* Some sub-categories like Tables show negative profit

## 📈 Outcome

Helped understand sales trends and profit distribution across regions and categories.
