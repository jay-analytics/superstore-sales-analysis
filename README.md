# 📊 Sales Data Analysis Project

## 🎯 Objective

The objective of this project is to analyze sales data and identify key factors affecting revenue and profitability across regions, categories, and products.

---

## 🛠️ Tools & Technologies

* Python (Pandas, Matplotlib)
* Excel (Data Cleaning)
* Power BI (Dashboard)
* Jupyter Notebook

---

## 📂 Dataset

Superstore Sales Dataset containing 5900+ records including sales, profit, region, category, and customer details.

---

## 🧹 Data Cleaning

* Removed empty columns
* Handled missing values and errors (#N/A → 0)
* Converted date columns to proper format
* Ensured correct data types

---

## 📊 Key Analysis Performed

### 🔹 Region Analysis

* Compared sales and profit across regions
* Calculated profit margins

### 🔹 Category & Sub-Category Analysis

* Identified loss-making and high-profit categories

### 🔹 Product-Level Analysis

* Found top loss-making products

### 🔹 Returns Analysis

* Evaluated impact of returned orders on overall profitability

---

## 💡 Key Insights

* South region has lowest profit margin despite good sales
* Furniture category generates low profit compared to others
* Sub-categories like Tables and Supplies are loss-making
* Copiers and Phones generate highest profit
* Returned orders contribute very little to total profit

---

## 🎯 Business Recommendations

* Focus on high-profit categories like Technology
* Reduce losses in Furniture category
* Improve strategy in low-performing regions
* Minimize product returns to improve profitability

---

## 📌 Conclusion

This project demonstrates how data-driven insights can help improve business performance and decision-making.
